#  Insurance Analytics Dashboard | Power BI

## Project Overview
This project presents an **interactive Insurance Analytics Dashboard** developed using **Power BI** to analyze insurance policy, customer, and claims data.  
The dashboard helps stakeholders monitor business performance, identify risk patterns, and make data-driven decisions efficiently.

This project is part of my **Data Analyst portfolio** and demonstrates real-world analytical and visualization skills.
 
---

##  Project Objectives
- Analyze insurance policies and customer data
- Track premium collection and claims trends
- Identify high-risk and high-value customers
- Provide actionable insights through KPIs and visuals

---

##  Tools & Technologies Used
- **Power BI**
- **Power Query (ETL)**
- **DAX (Data Analysis Expressions)**
- **Excel / CSV Dataset**

---

##  Dataset Description
The dataset includes:
- Customer demographic information  
- Insurance policy details  
- Premium amounts  
- Claims data  
- Policy status (active/inactive)  

> Dataset is used for learning and portfolio demonstration purposes.

---

##  Data Cleaning & Transformation
- Removed duplicate and missing values  
- Standardized column formats  
- Created calculated columns and measures using DAX  
- Built optimized data model relationships
